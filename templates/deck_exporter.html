How to Use

  1. After completing your draft on Draftmancer, export your deck using their export feature
  2. Copy the exported deck text and paste it into the Draftmancer Export box below
  3. Click Convert to ArkhamDB Format to transform the deck
  4. Copy the result from the ArkhamDB Import box
  5. Go to ArkhamDB's My Decks > Import Deck > Copy/paste then paste from the ArkhamDB Import section

Note: Only the main deck will be converted. Any cards after the first empty line (sideboard) will be ignored as they should not be uploaded to ArkhamDB.
